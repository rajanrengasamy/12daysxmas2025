# Art Prompts - Quick Guide

**Where to generate:** [gemini.google.com](https://gemini.google.com) or [aistudio.google.com](https://aistudio.google.com)

---

## How to Use

1. Open a prompt file (e.g., `door-01.md`)
2. Copy the text inside the ``` code block ```
3. Paste into Gemini
4. Generate 2-3 versions, pick your favorite
5. Save with the filename suggested in the file

1. **Start with:** `bg-01-cabin.md` (your main background)
2. **Then:** `frame-polaroid.md` (for your photos)
3. **Then:** All 12 doors in order
4. **Optional:** UI elements if you have time

## Tips

- Generate 2-3 versions of each image and pick the best
- If images look too blue/cold, the prompt already includes "warm" - try regenerating
- Save all images to an `assets` folder in your project
